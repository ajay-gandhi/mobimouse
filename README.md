# mobimouse

> Use your mobile device as a mouse + keyboard

Use your mobile device as a quick-and-dirty mouse + keyboard.

```bash
cd mobimouse/
npm install
cd server/
bash compile.sh
node app.js
```

Open the URL on your phone and go to work!
